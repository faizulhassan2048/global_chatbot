# 🌍 Global Free Chatbot

A completely free, multilingual chatbot that runs on your computer.

## Features
- ✅ 100% Free (uses Groq's free API)
- ✅ Works in 20+ languages
- ✅ Answers almost anything
- ✅ Runs locally on your computer
- ✅ Remembers past questions
- ✅ Beautiful Streamlit interface

## Quick Setup (5 Minutes)

### Step 1: Install Python
Download Python from python.org (version 3.9 or higher)

### Step 2: Get a Free Groq API Key
1. Go to https://console.groq.com
2. Sign up with email (free)
3. Click "Create API Key"
4. Copy the key

### Step 3: Setup Project
1. Open VS Code
2. Open Terminal (View → Terminal)
